## 1:
```go
func moveZeros(nums []int) []int {
    p1 := 0
    p2 := 0

    for p1 < len(nums) {
        if nums[p1] != 0 {
            nums[p1], nums[p2] = nums[p2], nums[p1]
            p1++
        }
        
        p2++
    }

    return nums
}
```
Ошибка: Программа выполнялась слишком долго и превысила лимит по времени

Я по не внимательности в условии цикла указал p1 (медленный указатель) вместо быстрого указателя p2, ведь задачу я решал методом "быстрый и медленный указатель", где p2 двигается каждый итератор, он быстрый, а следовательно по нему и ориентируемся. А ошибка превысила лимит скорей всего из-за того, что помимо этой ошибки я ещё вижу ошибку по условию, где я по медленному указателю опять проверяю не нулевой элемент, что опять же неправильно, мы должны по быстрому указателю искать, где ненулевой элемент, тот и ставить на место, где медленный указатель.
