## Решение:
```go
func compare(s string, t string) bool {
    clearS := clear(s)
    clearT := clear(t)

    p1 := 0
    p2 := 0

    for p1 < len(clearS) && p2 < len(clearT) {
        if clearS[p1] != clearT[p2] {
            return false
        }
        p1++
        p2++
    }

    return len(clearS) == len(clearT)
}

func clear(anyString string) []rune {
    result := make([]rune, 0, len(anyString))

    for _, r := range anyString {
        if r != '#' {
            result = append(result, r)
        } else if len(result) > 0 {
            result = result[:len(result)-1]
        }
    }

    return result
}
```

## Оценка по времени:
O(max(n, m)) - где n - размер входной строки s, m - размер входной строки t.

Объяснение: Для очистки строк я два раза прохожу по каждой строке, а строки могут быть разными, поэтому самое длинное прохождение по самой длинной строке

## Оценка по памяти:
O(max(n, m)) - где n - размер входной строки s, m - размер входной строки t.

Объяснение: аллоцирую два слайса в два захода, поэтому беру самый длинный и считаю большую память по нему

## Описание решения:
Я создал функция отчистки строк, как будто пройдена тектсовый редактор, то есть убрал по условию все символы перед решётками: просто итерировался по строке, каждую руну проверял на решётку, если не решётка, то аппендил в результатирующий чистый слайс рун, если решётка и при этом результатирующий слайс не пустой, то я срезал последний элемент из результатирующего слайса, либо ничего не делаю. Затем беру два указателя, применяю паттерн каждому по указателю и запускаю цикл с условием, что первый указатель меньше, чем длина первого слайса И второй указатель меньше, чем длина второго слайса, а внутри итерации проверяю, что если руны не равны по этим указателям, то сразу выкидываем false, завершая функцию. Если равны, то ничего не делаем, дальше итерируемся. А если цикл весь заканчивается, то тут два варианта: либо все руны были равны, либо какой-то указатель вышел за пределы, и, чтобы вернуть правильный результат сравнения, я делаю проверку, что длины слайсов равны.
